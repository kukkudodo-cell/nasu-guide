# 那須ハイランドパークガイド（iPhone用ガイドアプリ）

那須ハイランドパーク非公式ガイド。実際の園内マップ・のりもの・身長判定（115cm）・レストラン・おすすめルート・乗った記録を1つにまとめたオフライン対応アプリ。

## iPhoneに入れる（GitHub Pages 経由・所要5分）

### 1. GitHubにアップロード
1. github.com でアカウントを作る（無料）
2. 新しいリポジトリを作る（例：`nasu-guide`）
3. 下記 **8ファイル ＋ `img` フォルダ** を丸ごとアップロード（`_work` フォルダは不要）
   - `index.html`
   - `manifest.webmanifest`
   - `sw.js`
   - `map.jpg` … 地図の背景画像
   - `icon-180.png` / `icon-192.png` / `icon-512.png` / `apple-touch-icon.png`
   - `img/` フォルダ … のりもの・食事の写真45枚（`a1.jpg`〜、`rA.jpg`〜）

> `map.jpg` と `img/` の写真は那須ハイランドパーク公式サイトから取得・加工したもの。**自分・家族がiPhoneで使う分は問題なし**。全世界に公開されるPublicリポジトリで配信すると公式画像の再配布になるため、気になる場合はリポジトリを **Private** にして自分のiPhoneから開く。画像を外しても地図はイラスト模式図に、写真欄は自動で非表示に切り替わる（アプリは壊れない）。

### 2. Pagesを有効化
1. リポジトリの **Settings → Pages**
2. Source を **Deploy from a branch**、Branch を **main / (root)** にして Save
3. 1〜2分で `https://<ユーザー名>.github.io/nasu-guide/` が発行される

### 3. ホーム画面に追加
1. iPhoneの **Safari** で上記URLを開く
2. 下の **共有ボタン**（□に↑）をタップ
3. **ホーム画面に追加** を選ぶ
4. 右上の **追加** をタップ

観覧車アイコンのアプリになり、電波が弱い園内でもオフラインで起動する。

## 使い方
- **🗺️ マップ** … 実際の園内マップに、身長115cmで乗れるか色分けしたピンを重ねて表示（🟢のれる／🟡付き添いで／🔴のれない）。ピンをタップで詳細。ドラッグで移動・二本指で拡大。✓は乗った記録。
- **🎢 のりもの** … 公式写真つきカード。カテゴリ別・「115cmで乗れるものだけ」「まだ乗ってない」フィルタ。各カードの「📍地図で見る」で該当ピンへジャンプ。おすすめ度★つき。
- **🍴 食事** … 公式写真つき。レストラン6店のメニュー・価格・営業状況。
- **🚶 ルート** … 115cmキッズ王道／雨の日／絶叫 の3コース。
- **ℹ️ 情報** … 7/4(土)の営業・料金・駐車場・持ち物・ホーム画面追加手順。
- **✓ 乗った記録** … 詳細やカードの「のったにする」で記録すると、地図のピンに✓が付き「のった X/39」で進捗表示。iPhone内に保存され、閉じても残る（記録リセットで消去）。
- **👣 足あと線** … 乗った順にピンをオレンジの線でつなぎ、番号（1,2,3…）付きで当日の動線を地図に描く。回った軌跡が残る。
- 右上の **身長ピル** をタップすると身長を変更でき、全判定が切り替わる（初期値115cm）。
